Matricula en macOS Sierra

NOTA ACLARATORIA IMPORTANTE

Es importante que lean estas instrucciones desde sus computadoras y detenidamente. Si las instrucciones de abajo (después de esta nota aclaratoria) no te funcionan tendras que entrar el siguente comando cada vez que quieras conectarte a rumad.uprm.edu:

$ ssh -oKexAlgorithms=+diffie-hellman-group1-sha1 -oHostKeyAlgorithms=+ssh-dss estudiante@rumad.uprm.edu
El simbolo $ indica el prompt de comando en tu Mac. Todo lo que está después de $ es un comando para entrar (copy/paste). No lo incluyas en tus comandos.

Oprime Return. Su Mac debe conectarse a rumad.uprm.edu.

Si necesitan ayuda, mi email es {{ site.email }}.

NO SOY RESPONSABLE POR CUALQUIER DAÑO CAUSADO A SU COMPUTADORA.

Usen copy/paste para los comandos.

Primero

Abre un nuevo window de Terminal. En Terminal, haste seguro que estás en el directorio correcto. Escribe lo siguiente: 

$ pwd
Oprime Return. Debería salir lo siguente:

$ pwd
/Users/YOU

En sus computadoras, YOU será el nombre de tu usuario.

Si el resultado del comando pwd fue /Users/YOU significa que estás en el directorio correcto. Si tu resultado no fue el anterior, estás en el directorio incorrecto. Escribe el siguente comando:

$ cd

Oprime Return. Vuelve y corre el comando pwd y verifica que llegaste al directorio correcto.

Confirmando que estás en el directorio correcto, escribe el siguiente comando: 

$ open .ssh/config

Oprime Return. Si el Terminal te dice que ese archivo no existe, escribe el siguiente comando: 

$ touch .ssh/config

Oprime Return. Luego escribe escribe el comando anterior y oprime Return.

Es posible que el directorio .ssh/ no exista en tu Mac y el Terminal muestre el siguiente error: 

.ssh/: No such file or directory

En este caso, tendrás que crear el directorio. Escribe el siguiente comando: 

$ sudo mkdir .ssh

El prompt te va a pedir tu password de usuario.

Password: |

El password no va aparecer en la pantalla como tal, pero si está funcionando. Cuando escribas el password oprime Return.

Luego escribe estos comandos en orden:

$ touch .ssh/config

Oprime Return.

$ open .ssh/config

Oprime Return una vez más.

Segundo

Tu Mac debe haber abierto una nueva ventana de TextEdit con el documento config. Para ti, este archivo debe de estar vacio.

En este documento config, copy/paste este bloque de texto al archivo:

Host rumad.uprm.edu
      KexAlgorithms +diffie-hellman-group1-sha1
      HostKeyAlgorithms +ssh-dss

Tu config debe verse como en el screenshot de abajo. Asegurate que luzca de la misma forma que en el screenshot.

Tercero

Guarda tus cambios usando la combinación de teclado Command+S o navegando a File -> Save en TextEdit.

Conectate a rumad.uprm.edu usando el siguiente comando: 

$ ssh estudiante@rumad.uprm.edu

Oprime Return. Si todo está funcionando bien, deberias ver el prompt para entrar el password de estudiante. Oprime Return ya que estudiante no tiene password.

Cuarto

¡ÉXITO! Si seguistes las instrucciones correctamente, tu Terminal mostrará el menu del Sistema Estudiantil Colegial.
