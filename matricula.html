Matricula en macOS Sierra

NOTA

Es importante que lean estas instrucciones desde sus computadoras. El código tiene problemas de layout en móviles. Si las instrucciones de abajo no te funcionan, vas a tener que usar una forma menos permanente i.e. vas a tener que escribir un comando largo cada vez que quieras conectarte. Es recomendado que trates de hacer las instrucciones de abajo. Si no funciona, tendras que usar este comando:

ssh -oKexAlgorithms=+diffie-hellman-group1-sha1 -oHostKeyAlgorithms=+ssh-dss estudiante@rumad.uprm.edu

Si necesitan ayuda, mi email es {{ site.email }}.

NO SOY RESPONSABLE POR CUALQUIER DAÑO CAUSADO A SU COMPUTADORA.

Usen copy/paste para los comandos.

Primero

Abre un nuevo window de Terminal. En Terminal, escribe 

open .ssh/config

Si el Terminal te dice que ese archivo no existe, escribe primero 

touch .ssh/config
y luego el comando anterior.

Segundo

Luego que corras ese comando, tu Mac debe haber abierto un nuevo window de TextEdit. Para ti, debe de estar vacía. En mi caso no porque ya apliqué el arreglo para conectarse a rumad.uprm.edu.

En ese documento de TextEdit, vas a darle copy and paste a este parcho de texto:

Host rumad.uprm.edu
      KexAlgorithms +diffie-hellman-group1-sha1
      HostKeyAlgorithms +ssh-dss

Tu file debe verse como en el screenshot de abajo. Asegurate que tenga el formato exacto o quizás tengas problemas.

Tercero

Cuando hayas hecho lo anterior, guarda el archivo usando Command + S o navegando a File > Save en TextEdit.

Luego trata de conectarte a rumad.uprm.edu como de costumbre, usando 

ssh estudiante@rumad.uprm.edu
Si todo está funcionando bien, deberias ver el prompt para entrar el password para estudiante. Presiona enter (estudiante no tiene password.)

Cuarto

ÉXITO! Si hiciste todo bien, deberias ver el menu normal de Matrícula.

Espero que consigan las clases que necesitan.
