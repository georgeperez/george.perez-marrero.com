Matrícula en macOS Sierra

Read the English version {{ site.url }}/matricula

Nota Aclaratoria

Es importante que leas estas instrucciones desde tu computadora y detenidamente. Si las instrucciones de abajo (después de esta nota aclaratoria) no te funcionan tendras que entrar el siguente comando cada vez que quieras conectarte a rumad.uprm.edu:

ssh -oKexAlgorithms=+diffie-hellman-group1-sha1 -oHostKeyAlgorithms=+ssh-dss estudiante@rumad.uprm.edu

Oprime Return.

Usando el comando anterior, tu Mac debe conectarse a rumad.uprm.edu. Si necesitan ayuda, mi email está al final de esta página.

No soy responsable de ningún daño que ocurra con el uso de estas instrucciones.

Usa copy/paste para los comandos.

Primero

OJO

El símbolo $ indica el prompt de comando en tu Mac. Todo lo que está después del símbolo $ es un comando para entrar (copy/paste). NO lo incluyas en tus comandos.

Abre una nueva ventana de Terminal. En Terminal, haste seguro que estás en el directorio correcto. Escribe el siguiente comando: 

$ pwd
Oprime Return. Debería salir lo siguente:

$ pwd
/Users/YOU

On your computer, YOU will be your username.

Si el resultado del comando pwd fue /Users/YOU significa que estás en el directorio correcto. Si tu resultado no fue el anterior, estás en el directorio incorrecto. Escribe el siguente comando:

$ cd

Oprime Return. Vuelve y corre el comando pwd y verifica que llegaste al directorio correcto.

Confirmando que estás en el directorio correcto, escribe el siguiente comando: 

$ open .ssh/config

Oprime Return. Este command debe haber abierto una nueva ventana de TextEdit.

Posibles problemas y sus arreglos

1. "The file /Users/your-username/.ssh/config does not exist"

Si tu ventana de Terminal muestra el siguiente error: 

The file /Users/your-username/.ssh/config does not exist
Significa que el archivo config no existe en tu Mac. Para arreglar esto, escribe el siguiente comando para crear el archivo: 
$ touch .ssh/config

Oprime Return. Luego, escribe el comando anterior: 

$ open .ssh/config
Oprime Return otra vez.

2. "No such file or directory"

It's possible that the .ssh/ directory does not exist on your Mac. Your Terminal window will show the following error: 

.ssh/: No such file or directory

En este caso, tienes que crearlo. Escribe el siguiente comando: 

$ sudo mkdir .ssh

OJO

La parte sudo del comando anterior eleva tu Terminal a usar privelegios de administrador.

El prompt pedirá tu password de usuario.

Password:

Escribe tu password de usuario. El password no va aparecer el pantalla, pero sí se está escribiendo. Cuando termines de escribir tu password, oprime Return.

Luego de todo eso escribe los siguientes comandos en orden:

$ touch .ssh/config

Oprime Return.

$ open .ssh/config

Oprime Return una vez más.

Segundo

Tu Mac debe haber abierto una nueva ventana de TextEdit con el documento config. Para ti, este archivo debe de estar vacío.

En este documento config, copy/paste el siguiente bloque de texto en el archivo:

Host rumad.uprm.edu
      KexAlgorithms +diffie-hellman-group1-sha1
      HostKeyAlgorithms +ssh-dss

Tu config debe lucir como el siguiente screenshot. Asegurate que luzca de la misma forma que el screenshot.

Tercero

Guarda tus cambios usando la combinación de teclado Command+S o navegando a File -> Save en TextEdit.

Conectate a rumad.uprm.edu usando el siguiente comando: 

$ ssh estudiante@rumad.uprm.edu

Oprime Return. Si todo está funcionando bien, deberias ver el prompt para entrar el password de estudiante. Oprime Return ya que estudiante no tiene password.

Cuarto

¡ÉXITO! Si seguistes las instrucciones correctamente, tu Terminal mostrará el menu del Sistema Estudiantil Colegial.

Por favor envia cualquier pregunta o recomendación a mi email.
